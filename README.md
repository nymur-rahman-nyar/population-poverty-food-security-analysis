# Project Description

Stack :
pandas, Jupyter Notebook

Data Used :

**United Nations – World Population Prospects (2024 Revision)**

* Dataset: Demographic Indicators (1950–2100, medium variant)
* Indicators used include total population, population density, median age, and natural population change.
* Source: UN Department of Economic and Social Affairs, Population Division.
* Path : data/raw/WPP2024_Demographic_Indicators_Medium.csv


**World Bank – World Development Indicators (WDI)**

* Dataset: World Development Indicators
* Indicator used: Poverty headcount ratio at societal poverty line (% of population)
* Description: Measures the share of the population living below the World Bank’s societal poverty line for Bangladesh
* Data characteristics: Non-annual observations, reported for selected survey years
* Source: World Bank DataBank
* Path (raw): `data/raw/ad695f8c-9b6d-4f65-b85c-20be7331858c_Data.csv`
* Path (processed):`data/processed/bangladesh_poverty_clean.csv`
